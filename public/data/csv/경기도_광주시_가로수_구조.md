# 경기도 광주시 가로수(공간정보) 데이터 구조

- **폴더 위치**: `dist/data/csv/경기도 광주시_가로수(공간정보)_20251120/`
- **형식**: Shapefile (포인트) — CSV가 아닌 공간 데이터 세트
- **좌표계**: Korea 2000 / Korea Central Belt 2010 (EPSG:5186, 미터 단위) → 웹 지도 사용 시 WGS84(위경도)로 변환 필요
- **데이터 기준**: 2025-11-20 제공, 2025-09-15 내보내기

- **참고**: 엑셀 파일이므로 변환 스크립트에서는 `xlsx` 패키지 등으로 읽거나, 수동으로 CSV로 저장해 사용 가능

---

## 지도/트리맵에 쓰기 위해 필요한 작업

1. **Shapefile → GeoJSON(또는 CSV)**  
   - SHP + DBF를 읽어서 **도형(포인트)** 과 **속성(TRE_CDE, BJD_CDE, LOC_DES 등)** 을 합친 형태로 변환  
   - 좌표는 **EPSG:5186 → WGS84** 변환 필요 (GDAL `ogr2ogr` 또는 Node에서 `proj4` 등 사용)

2. **수종 코드 해석**  
   - `코드값정보(가로수).xlsx`에서 TRE_CDE 매핑 확인 후, 수종별 그루수·마커 표시에 활용

3. **앱 데이터 형식 맞추기**  
   - 기존 부산/전주/정읍/완주와 같이 **구간(세그먼트) 단위 집계** 또는 **포인트 그대로** 중 앱 스펙에 맞게 JSON 생성 (예: `BusanSegment` 형태로 구간별 집계하거나, 포인트 GeoJSON으로 표시)
